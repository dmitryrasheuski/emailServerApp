Для работы данного приложения необходимо:
    * "Docker";
    * почтовый аккаунт(email и пароль);
    * в настройках почтового аккаунта должен быть включен доступ по протоколу IMAP;
    * (gmail.com) в настройках google аккаунта открыть доступ к аккаунту ненадежным приложениям.
    
Запуск приложения:
    * в папке приложения найти и открыть файл environment.txt, в котором необходимо прописать
      email почтового аккаунта и пароль от него:
            EMAIL=someEmail@some.xxx
            PASS=some1234
    * из папки с приложением в командной строке прописать:
            docker-compose up -d

Остановка приложения осуществляется из командной строки командой: docker-compose down.

Работа с приложением осуществляется в браузере по адресу: localhost:8084
При перезапуске ОС приложение должно включаться самостоятельно.
Для проверки состояния приложения необходимо в командной строке прописать: docker ps.
